Как зафиксировать «снимок» проекта перед крупными правками
============================================================

1) Git (лучший вариант)
   - Установи Git, в корне прота:  git init
   - git add -A
   - git commit -m "Стабильное состояние перед загрузкой фото в админке"
   - По желанию: git tag stable-2026-04-05

2) Без Git
   - Скопируй всю папку проекта в архив / на флешку (ZIP в проводнике).

3) После этого деплоя с загрузкой фото
   - На VPS после git pull: в папке backend выполни npm install и перезапусти процесс Node.
   - Убедись, что папка backend/uploads/products существует и процесс может в неё писать.
   - В Nginx добавь проксирование пути /uploads на бэкенд (см. backend/README.md).

Файлы загрузок в репозиторий не коммитятся (см. .gitignore), бэкап БД и папки uploads на сервере — отдельно при необходимости.
