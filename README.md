# Dramma

Магазин косметики: каталог, поиск, карточка товара, корзина, избранное, заказ, профиль, админка.

**Стек:** React, TypeScript, Vite, Zustand · Node.js, Express, Prisma, MySQL · JWT (access/refresh), код на почту через Resend.

Фронт в корне репозитория, API в `backend/`. Запуск и переменные окружения — в `backend/README.md`.
